# Points Explorer

A React application for finding flight destinations based on your credit card points.

## Features

- Select from multiple credit cards and rewards programs
- Filter destinations by points, tax limits, cabin class, and more
- Find the best airline transfer partners for your trip
- Compare point valuations across different programs

## Demo

Visit the live app at [points-explorer.vercel.app](https://points-explorer.vercel.app)

## Installation

```bash
git clone https://github.com/yourusername/points-explorer.git
cd points-explorer
npm install
npm start
```

## Technologies Used

- React.js
- Tailwind CSS
- Lucide React for icons
- Recharts for data visualization

## Deployment

This app is configured for easy deployment on Vercel.

## Adding More Data

The app uses sample data for demonstrations. In a production environment, you would want to connect to:

- The Points Guy API for current point valuations
- Award Wallet Developer API for real-time availability
- FlightConnections API for route network data
